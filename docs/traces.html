About these traces: All execution traces were recorded using LangSmith during security validation experiments described in Section VI of the paper. Each trace contains complete execution details including LLM reasoning steps, tool invocations, token minting requests, and timing measurements.

Reproducibility: Traces are provided in two formats: (1) archived HTML snapshots for permanent access, and (2) live interactive traces on LangSmith for real-time exploration (if project is public).
